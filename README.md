# دیوار دوم - پلتفرم خرید و فروش کالاهای دست دوم

پلتفرم آنلاین برای خرید و فروش محصولات دست دوم که به کاربران اجازه می‌دهد کالاهای دست دوم خود را آگهی کنند و دیگران بتوانند آن‌ها را خریداری کنند.

## ویژگی‌های اصلی

- مدیریت کاربران (ثبت‌نام، ورود، احراز هویت)
- ثبت آگهی فروش با جزئیات کامل و تصاویر
- جستجو و فیلتر پیشرفته محصولات
- سیستم چت و مذاکره بین خریدار و فروشنده
- سیستم نظرات و امتیازدهی
- بخش پشتیبانی و حل اختلافات
- داشبورد مدیریتی
- مدیریت سفارش و رهگیری ارسال

## نصب و راه‌اندازی

1. نصب وابستگی‌ها:
```
npm install
```

2. اجرای برنامه در حالت توسعه:
```
npm start
```

3. ساخت نسخه نهایی برای انتشار:
```
npm run build
```
